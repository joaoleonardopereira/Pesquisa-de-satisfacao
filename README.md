# 📊 Pesquisa de Satisfação

Este projeto foi desenvolvido como parte de um exercício prático do curso técnico em **Desenvolvimento de Sistemas**, com o objetivo de simular uma pesquisa de opinião.

A aplicação foi construída em Python e tem como finalidade coletar e analisar a satisfação dos clientes em relação ao atendimento prestado.

---

## 🚀 Funcionalidades

* Coleta de dados de até 50 entrevistados
* Registro de:

  * Nome
  * Idade
  * Avaliação do atendimento
* Classificação das respostas em:

  * **1 - Excelente**
  * **2 - Bom**
  * **3 - Ruim**
* Contabilização automática das respostas
* Exibição do resultado final da pesquisa

---

## 🧠 Lógica do Programa

O sistema utiliza:

* Estrutura de repetição `for` para realizar múltiplas entrevistas
* Estruturas condicionais (`if`, `elif`) para classificar as respostas
* Variáveis contadoras para armazenar os resultados
* Entrada de dados via teclado (`input()`)

---

## 🧪 Testes

Durante o desenvolvimento, foram realizados testes com **10 entrevistados**, permitindo validar o funcionamento do sistema antes da execução completa com 50 usuários.

---

## 📈 Saída Esperada

Ao final da execução, o programa exibe:

* Quantidade de respostas **EXCELENTE**
* Quantidade de respostas **BOM**
* Quantidade de respostas **RUIM**
